## 我们需要什么样的帮助

普通人学编程，最难的有亮点： 1 陌生枯燥的概念 2 不知道如何一步步开发出自己所需的工具

“小欣学Python"面临的挑战就是 用风趣易懂的语言介绍这些编程相关的概念 和 构建出让读者能够渐进式开发出解决身边的实际问题项目关卡；
在此，我们呼吁有能力的朋友加入我们，提供能让你爸妈都能理解的编程概念 和 精心设计练小孩都能通关的编程关卡。
